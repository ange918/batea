# 🟢 PROMPT REPLIT AGENT — NIGHTGLOW (NEXT.JS + HEROICONS)

---

## CONTEXTE DU PROJET

Crée un site web complet et immersif pour **NightGlow**, une marque africaine de design phosphorescent fondée à Cotonou, Bénin, en 2025 par GBETOENONMON Sèdami Mike. NightGlow crée des vêtements, accessoires et objets décoratifs qui brillent dans le noir grâce aux matériaux phosphorescents (strontium aluminate), sans électricité, sans pile, sans artifice.

**Slogan** : *"L'Afrique brille. Même dans le noir."*
**Tagline produit** : *"Let There be Light"*

Le site doit être sombre, luxueux, africain et visuellement saisissant. Fond noir total, accents vert fluo, typographie ultra-bold, effets de lueur phosphorescente.

---

## STACK TECHNIQUE

- **Next.js 14** avec App Router
- **TypeScript**
- **Tailwind CSS**
- **@heroicons/react v2** pour toutes les icônes — aucune autre librairie d'icônes
- **Framer Motion** pour toutes les animations
- **next/font** pour charger : `Bebas Neue` (titres) + `DM Sans` (corps de texte)

- **Bebas Neue** → tous les `<h1>`, `<h2>`, grandes métriques chiffrées, numéros de section

---

## EFFETS VISUELS GLOBAUX

- **Glow phosphorescent** : `text-shadow` vert sur les titres importants et chiffres clés
- **Lueur sur les cartes** : `box-shadow` halo vert subtil, visible au hover
- **Barre verticale verte** de 4px sur le bord gauche du hero
- **Particules flottantes** dans le hero : petits points verts qui montent/descendent avec Framer Motion
- **Animations au scroll** : chaque section fait un `fadeInUp` (opacity 0→1, translateY 30→0) via Framer Motion `whileInView`
- **Compteurs animés** : les stats chiffrées s'incrémentent de 0 à leur valeur dès qu'elles entrent dans le viewport
- **Scrollbar custom** : 4px de large, verte
- **Curseur custom** : petit point vert lumineux sur desktop
- **Smooth scroll** sur toute la page

## RÈGLES ABSOLUES

1. Toutes les icônes viennent **uniquement** de `@heroicons/react/24/outline` ou `/24/solid`
2. **Jamais de fond blanc** — le site est entièrement noir
3. **Bebas Neue** pour tous les titres et grandes métriques sans exception
4. **Framer Motion `whileInView`** sur chaque section pour l'animation au scroll
5. Site **100% responsive** — mobile-first, breakpoints Tailwind sm/md/lg/xl
6. Les compteurs de la section Marché s'animent avec un hook `useCountUp` personnalisé
7. Le toggle Jour/Nuit de la Collection est géré par `useState` React
8. Toutes les cartes ont un `hover` avec halo vert (`shadow-glow`)
9. Smooth scroll sur tous les liens ancres de navigation
10. Aucune image externe — visuels en SVG pur dessinés en code ou placeholders CSS stylisés
